# BaseGrid Frontend

This is the Next.js frontend for the BaseGrid platform. It uses React, TailwindCSS, and Wagmi for interacting with the Base network.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Configuration

Make sure to configure the `.env.local` file with your specific variables.
